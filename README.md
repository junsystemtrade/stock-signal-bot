# 米国株シストレ通知システム

本システムは、米国株を対象にした自動トレードシグナル生成およびDiscord通知システムです。指定銘柄のストキャスティクス指標を利用して買いシグナルを判定し、日本時間で毎日18:05に通知します。

## 概要

- 米国株（例: JMIA、NU）を対象  
- yfinance から株価データを取得（米国時間）  
- 過去1年間のデータを保持  
- ストキャスティクス（%K、%D）計算  
- 25以下で買いシグナル発生  
- 前日のシグナルを翌日始値で約定  
- 保有株数、評価額、損益を集計  
- Discord通知（日本時間18:05）
- 土曜：週次レポート送信（米国市場月〜金）
- 日曜・祝日：前営業日集計通知

---

## 機能

- 米国株の株価取得（米国市場営業日）
- ストキャスティクス指標計算
- 新規買いシグナル判定
- 前日のシグナル約定（Buy_Price更新）
- CSV更新 (`trade_history.csv`)
- Discord通知作成・送信
- 土曜週次レポート生成
- 日曜・祝日：前営業日のレポート通知

```bash
